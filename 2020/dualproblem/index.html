数学系列——对偶问题

原始问题

考虑一般的带约束优化问题：

minx∈Rnf(x)s.t.ci(x)≤0i=1,2,...,khj(x)=0j=1,2,...,l\mathop{min}\limits_{x \in R^n} f(x)\\ s.t. \quad c_i(x) \le 0 \quad i = 1,2,...,k\\ \quad \quad h_j(x) = 0 \quad j = 1,2,...,l x∈Rnmin​f(x)s.t.ci​(x)≤0i=1,2,...,khj​(x)=0j=1,2,...,l

为了将上述约束写到待求解的式子中，我们可以引入拉格朗日函数，得到：

L(x,α,β)=f(x)+∑i=1kαici(x)+∑j=1lβjhj(x)L(x, \alpha, \beta) = f(x) + \sum_{i=1}^k \alpha_i c_i(x) + \sum_{j=1}^l \beta_j h_j(x) L(x,α,β)=f(x)+i=1∑k​αi​ci​(x)+j=1∑l​βj​hj​(x)

其中 αi≥0\alpha_i \ge 0αi​≥0

然后，对上面式子求极大，即

θp(x)=maxα,β,αi≥0L(x,α,β)\theta_p (x) = \mathop{max}\limits_{\alpha, \beta, \alpha_i \ge 0} L(x, \alpha, \beta) θp​(x)=α,β,αi​≥0max​L(x,α,β)

容易证明，在x满足原问题约束的条件下，θp(x)=f(x)\theta_p (x) = f(x)θp​(x)=f(x)

  • 若存在一个x，不满足原始问题的约束条件，即ci(x)>0c_i(x) > 0ci​(x)>0 或者hj(x)≠0h_j(x) \neq 0hj​(x)​=0，则θp(x)\theta_p(x)θp​(x)的最大值一定可以取到无穷大。（取αi=∞\alpha_i = \inftyαi​=∞或者βjhj(x)=∞\beta_j h_j(x) = \inftyβj​hj​(x)=∞。
  • 反之，若x都满足原问题约束，则θp(x)\theta_p(x)θp​(x)取最大值时，αi\alpha_iαi​一定等于0。此时，θp(x)=f(x)\theta_p (x) = f(x)θp​(x)=f(x)

因此，带约束的原问题的等价问题（注意，不是对偶问题）为：

minxf(x)=minxθp(x)=minxmaxα,β,αi≥0L(x,α,β)\mathop{min}\limits_{x} f(x) = \mathop{min}\limits_{x} \theta_p(x) = \mathop{min}\limits_{x} \mathop{max}\limits_{\alpha, \beta, \alpha_i \ge 0} L(x, \alpha, \beta) xmin​f(x)=xmin​θp​(x)=xmin​α,β,αi​≥0max​L(x,α,β)

可见，原问题是一个先求极大，再求极小的问题。而求极大的过程中同样有约束αi≥0\alpha_i \ge 0αi​≥0的限制，此过程并不简单。所以，往往转换成对偶问题进行求解。

对偶问题

将原始问题求极大极小的问题转换为求极小极大的问题，即得到原始问题的对偶问题：

D(x)=maxα,β,αi≥0minxL(x,α,β)D(x) = \mathop{max}\limits_{\alpha, \beta, \alpha_i \ge 0} \mathop{min}\limits_{x} L(x, \alpha, \beta) D(x)=α,β,αi​≥0max​xmin​L(x,α,β)

上述问题第一步求解关于x的极小时，没有任何约束条件，可以很方便地求解。并且，对偶问题与原始问题有很强的关系，即

d∗=maxα,β,αi≥0minxL(x,α,β)≤minxmaxα,β,αi≥0L(x,α,β)=p∗d^* = \mathop{max}\limits_{\alpha, \beta, \alpha_i \ge 0} \mathop{min}\limits_{x} L(x, \alpha, \beta) \le \mathop{min}\limits_{x} \mathop{max}\limits_{\alpha, \beta, \alpha_i \ge 0} L(x, \alpha, \beta) = p^* d∗=α,β,αi​≥0max​xmin​L(x,α,β)≤xmin​α,β,αi​≥0max​L(x,α,β)=p∗

即，对偶问题的上限为原始问题的下限。

特别地，当满足某些条件（Slater条件）时，构成强对偶关系。此时，对偶问题的最优解与原始问题的最优解一致。
