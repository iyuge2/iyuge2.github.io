Linux服务端配置

  • 安装python，推荐使用Anaconda安装
  • 安装mysql服务
  1. sudo apt update
  2. sudo apt install mysql-server

远程连接

在Linux服务器上配置好了mysql之后，可以在本地下载一个mysql客户端，方便查看服务器端mysql数据库的具体情况。这里，我们使用了免费的Sequel Pro。

  • 修改bind-address

bind-address 的作用是告诉计算机，MySQL服务监听哪些ip地址，默认仅监听本地ip。

找到bind-address行，将其注释掉，或者改为0.0.0.0

  • 为用户授予远程登录权限

  • 修改本地Sequel Pro连接配置

这里使用SSH连接服务器，输入相关配置即可

python中Flask设置

  • 问题
  1. ModuleNotFoundError: No module named ‘MySQLdb’

环境中缺少mysqlclient，但直接用pip安装会报错。参考[Linux上安装mysqlclient]，先用命令sudo apt install libmysqlclient-dev，然后安装mysqlclient即可。
