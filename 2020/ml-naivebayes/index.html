机器学习系列—朴素贝叶斯

发表于2020-10-07|更新于2021-01-13|技术区
|字数总计:2.2k|阅读时长:9分钟|阅读量:

朴素贝叶斯算法是基于贝叶斯定理和特征条件独立性假设的分类方法。

基本原理

朴素贝叶斯算法是一种概率计算方法，目标是计算P(Y=ck∣X=x)P(Y=c_k | X=x)P(Y=ck​∣X=x)（后验概率），这也是所有分类算法的核心目标。考虑到在训练数据中，是可以得到P(X=xi∣Y=ck)P(X=x_i | Y=c_k)P(X=xi​∣Y=ck​)（条件概率）与P(Y=ck)P(Y=c_k)P(Y=ck​)（先验概率）的，所以，朴素贝叶斯算法就是研究如何利用后两个概率计算目标概率。

为了实现概率的转化，首先用到的就是贝叶斯定理:

P(Y∣X)=P(X∣Y)P(Y)P(X)P(Y|X) = \frac{P(X|Y)P(Y)}{P(X)} P(Y∣X)=P(X)P(X∣Y)P(Y)​

因为，对于所有的类别概率P(Y=ck∣X=x)P(Y=c_k | X=x)P(Y=ck​∣X=x)而言，P(X)P(X)P(X)是一个定值，因此只考虑分子即可。分子中P(Y)P(Y)P(Y)是容易得到的，难点在于如何计算P(X∣Y)P(X|Y)P(X∣Y)。通过公式

P(X∣Y=ck)=P(X(1)=x(1),...,X(n)=x(n)∣Y=ck),k=1,2,...,KP(X|Y=c_k) = P(X^{(1)} = x^{(1)}, ..., X^{(n)} = x^{(n)} | Y = c_k), k=1,2,...,K P(X∣Y=ck​)=P(X(1)=x(1),...,X(n)=x(n)∣Y=ck​),k=1,2,...,K

这个式子的参数量非常大，直接计算的代价很高，因此朴素贝叶斯引入了特征的条件独立性假设（即，不同特征之间完全独立）。在此假设下，有

P(X(1)=x(1),...,X(n)=x(n)∣Y=ck)=∏iP(X(i)=x(i)∣Y=ck)P(X^{(1)} = x^{(1)}, ..., X^{(n)} = x^{(n)} | Y = c_k) = \prod_i P(X^{(i)} = x^{(i)} | Y = c_k) P(X(1)=x(1),...,X(n)=x(n)∣Y=ck​)=i∏​P(X(i)=x(i)∣Y=ck​)

这个假设条件也是朴素贝叶斯中朴素两个字的由来。

因此，目标概率的计算公式为：

P(Y=ck∣X)=P(Y=ck)∏iP(X(i)=x(i)∣Y=ck)P(Y=c_k|X) = P(Y=c_k) \prod_i P(X^{(i)} = x^{(i)} | Y = c_k) P(Y=ck​∣X)=P(Y=ck​)i∏​P(X(i)=x(i)∣Y=ck​)

最后，朴素贝叶斯法将实例分到后验概率最大的类中：

y=argmaxckP(Y=ck)∏iP(X(i)=x(i)∣Y=ck)y = \mathop{argmax}\limits_{c_k} P(Y=c_k) \prod_i P(X^{(i)} = x^{(i)} | Y = c_k) y=ck​argmax​P(Y=ck​)i∏​P(X(i)=x(i)∣Y=ck​)

实际计算过程中，会利用对数变换将连乘的概率转换为连加的对数和

y=argmaxck{lnP(Y=ck)+∑ilnP(X(i)=x(i)∣Y=ck)}y = \mathop{argmax}\limits_{c_k} \{ln P(Y=c_k) + \sum_i lnP(X^{(i)} = x^{(i)} | Y = c_k)\} y=ck​argmax​{lnP(Y=ck​)+i∑​lnP(X(i)=x(i)∣Y=ck​)}

参数估计

为了计算后验概率，需要先得到条件概率和先验概率，意味着需要利用已知数据估计P(Y=ck)P(Y = c_k)P(Y=ck​)以及P(X(i)=x(i)∣Y=ck)P(X^{(i)} = x^{(i)} | Y = c_k)P(X(i)=x(i)∣Y=ck​)。这里介绍两类估计方法：

  • 极大似然估计

先验概率

P(Y=ck)=∑i=1NI(yi=ck)NP(Y = c_k) = \frac{\sum_{i=1}^N I(y_i = c_k)}{N} P(Y=ck​)=N∑i=1N​I(yi​=ck​)​

条件概率

P(X(j)=x(j)∣Y=ck)=∑i=1NI(X(j)=x(j),yi=ck)∑i=1NI(yi=ck)P(X^{(j)} = x^{(j)} | Y = c_k) = \frac{\sum_{i=1}^N I(X^{(j)} = x^{(j)}, y_i = c_k)}{\sum_{i=1}^N I(y_i = c_k)} P(X(j)=x(j)∣Y=ck​)=∑i=1N​I(yi​=ck​)∑i=1N​I(X(j)=x(j),yi​=ck​)​

  • 拉普拉斯平滑

上面那种方法在计算条件概率时可能会存在分母为0的情况。为了解决这个问题，容易想到的方法是在分母中增加一个常量，贝叶斯估计就是这么做的。
条件概率

P(X(j)=x(j)∣Y=ck)=∑i=1NI(X(j)=x(j),yi=ck)+λ∑i=1NI(yi=ck)+SiλP(X^{(j)} = x^{(j)} | Y = c_k) = \frac{\sum_{i=1}^N I(X^{(j)} = x^{(j)}, y_i = c_k) + \lambda}{\sum_{i=1}^N I(y_i = c_k) + S_i \lambda} P(X(j)=x(j)∣Y=ck​)=∑i=1N​I(yi​=ck​)+Si​λ∑i=1N​I(X(j)=x(j),yi​=ck​)+λ​

其中，SjS_jSj​指已知数据中X(j)X^{(j)}X(j)取值为x(j)x^{(j)}x(j)的个数，即Sj=∑i=1NI(X(j)=x(j))S_j = \sum_{i=1}^N I(X^{(j)} = x^{(j)})Sj​=∑i=1N​I(X(j)=x(j))。λ\lambdaλ是一个平滑常数，常取值为1（拉普拉斯平滑）。
为了迎合条件概率的变化，先验概率也要进行相应的平滑，即

P(Y=ck)=∑i=1NI(yi=ck)+λN+KλP(Y = c_k) = \frac{\sum_{i=1}^N I(y_i = c_k) + \lambda}{N + K \lambda} P(Y=ck​)=N+Kλ∑i=1N​I(yi​=ck​)+λ​

其中，K为类别数量。

引申-期望风险最小化准则

我们已经知道朴素贝叶斯法是后验概率最大化的结果，其实这个过程等价于0-1损失下的期望风险最小化的准则。
首先，定义一个0-1风险函数：

L(Y,f(x))={1,Y≠f(x)0,Y=f(x)L(Y, f(x)) = \begin{cases} 1, & Y \neq f(x) \\ 0, & Y = f(x) \end{cases} L(Y,f(x))={1,0,​Y​=f(x)Y=f(x)​

然后，期望风险为：

Rexp(f)=E[L(Y,f(x))]=EX∑k=1KL(ck,f(x))P(ck∣x)R_{exp}(f) = E[L(Y, f(x))] = E_X\sum_{k=1}^K L(c_k, f(x)) P(c_k | x) Rexp​(f)=E[L(Y,f(x))]=EX​k=1∑K​L(ck​,f(x))P(ck​∣x)

接下来，我们取期望风险最小化的结果作为分类结果f(x)f(x)f(x)，有：

f(x)=argminy∈Y∑k=1KL(ck,y)P(ck∣X=x)=argminy∈Y∑k=1KP(y≠ck∣X=x)=argminy∈Y(1−P(y=ck∣X=x))=argmaxy∈YP(y=ck∣X=x)\begin{aligned} f(x) &= \mathop{argmin}\limits_{y \in Y} \sum_{k=1}^K L(c_k, y) P(c_k | X = x) \\ &= \mathop{argmin}\limits_{y \in Y} \sum_{k=1}^K P(y \neq c_k | X = x) \\ &= \mathop{argmin}\limits_{y \in Y} (1 - P(y = c_k | X = x)) \\ &= \mathop{argmax}\limits_{y \in Y} P(y = c_k | X = x) \end{aligned} f(x)​=y∈Yargmin​k=1∑K​L(ck​,y)P(ck​∣X=x)=y∈Yargmin​k=1∑K​P(y​=ck​∣X=x)=y∈Yargmin​(1−P(y=ck​∣X=x))=y∈Yargmax​P(y=ck​∣X=x)​

所以，后验概率最大化准则等价于0-1损失下的期望风险最小化准则。

优缺点

优点

  • 分类准确率高，速度快
  • 对小规模数据表现很好，能够处理多分类任务，并且适合增量式训练（当数据量超出内存时，可以分批训练）

缺点

  • 对输入数据的表现形式敏感（离散、连续）
  • 对训练数据依赖大，如果训练数据误差较大，会对结果产生较大影响
  • 由于“朴素”的特点，在特征属性关联度较大的情况时，分类效果不好

面试点

1. 朴素贝叶斯与LR的区别

  • 朴素贝叶斯是生成模型，根据已有样本进行贝叶斯估计学习出先验概率P(Y)和条件概率P(X|Y)，进而求出联合分布概率P(XY),最后利用贝叶斯定理求解P(Y|X)， 而LR是判别模型，根据极大化对数似然函数直接求出条件概率P(Y|X)；
  • 朴素贝叶斯是基于很强的条件独立假设（在已知分类Y的条件下，各个特征变量取值是相互独立的），而LR则对此没有要求；
  • 朴素贝叶斯适用于数据集少的情景，而LR适用于大规模数据集。

2. 有了“朴素”的假设，为什么仍然可以取得较好的效果

对于分类任务，关心的不是概率的绝对值，而只是相对值。这种情况下，如果属性之间的依赖对所有类别的影响相同，或者可以相互抵消。这种情况下，条件独立性假设在降低计算开销的同时不会对分类结果产生不良影响。

3. 朴素贝叶斯有没有超参数

朴素贝叶斯没有超参数可调，不需要调参。对未知样本的预测值完全由训练数据决定。

4. 常见应用

文档分类、垃圾文件过滤、情感分析、推荐系统等。

5. 对异常值是否敏感

朴素贝叶斯对异常值不敏感，可以不用去除异常值。相反，异常值可以增强模型在预测过程中的泛化能力。

6. 朴素贝叶斯是高方差还是低方差模型

朴素贝叶斯是低方差模型。对于复杂模型，容易产生过拟合，这种情况下的偏差一般较小，方差较大。而朴素贝叶斯是一个简单模型，恰好相反，偏差较大，方差较小。
