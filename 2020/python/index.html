python中容易忽视的小知识点

python
技术区
发布日期:   2020-09-28

写在前面

本文不会记录python的详细知识，而只是记录笔者在回顾python官方文档时的笔记。

数组与字符串
  • 切片的越界索引会被自动处理
arr = [1,2,3,4,5]
arr[2:10] # 3,4,5
控制语句
  • 在循环里修改序列的值，可以先创建一份拷贝副本[:]
for w in words[:]:
    if len(w) > 5:
        words.insert(0, w)
  • 循环语句（不是if子句哦）可能带有一个else子句，它在循环结束后被执行，但是不会在循环被break语句终止时执行。
for n in range(1, 10):
    if n > 5:
        break
    print(n)
else:
    print("Over")
>>> 1 2 3 4 5
函数
  • 函数参数定义中，默认值只会执行一次，这条规则在默认值为可变对象（列表，字典等）时非常重要。
def f(a, L=[]):
    L.append(a)
    return a

print(f(1)) # [1]
print(f(2)) # [1,2]
print(f(3)) # [1,2,3]
  • 可变参数
    • *args: 接收一个包含除了已有行参列表以外的位置参数的元组的行参
    • **args: 接收一个字典，包含除了与已有行参相应的关键字参数以外的所有关键字参数
def cheeseshop(kind, *arguments, **keywords):
    for arg in arguments:
        print(arg)
    print("-" * 40)
    for kw in keywords:
        print(kw, ":", keywords[kw])
>>> cheeseshop("apple", "a1", "a2", client="John", shopkeeper="Bob")
  • 使用*运算符编写函数调用可以从列表或者元组中解包参数
args = [3,6]
list(range(*args)) # 等价于list(range(3,6))
  • 文档字符串规范

第一行简要概述对象的目的，然后空一个行，后面分段阐述必要说明

def function():
    """Do nothing, but document it.

    It doesn't do anything.
    """
    pass
>>> print(function.__doc__)
类
  • 实例变量用于每个实例的唯一数据，而类变量用于类的所有实例共享的属性和方法。
  • 数据属性会覆盖掉具有相同名称的方法属性。
  • python有两个内置函数可被用于继承机制：
    • isinstance()来检查一个实例的类型，isinstance(obj, int)
    • 使用issubclass()来检查类的继承关系：issubclass(bool, int)为True，因为bool是int的子类，但是issubclass(float, int)为False
  • 私有数据成员和方法的名字以双下划线__开头
迭代器与生成器
  • 迭代器

迭代器类中定义一个__iter__() 方法来返回一个带有__next__() 方法的对象。 如果类已定义了__next__()，则__iter__() 可以简单地返回self

class Reverse:
    """Iterator for looping over a sequence backwards."""
    def __init__(self, data):
        self.data = data
        self.index = len(data)

    def __iter__(self):
        return self

    def __next__(self):
        if self.index == 0:
            raise StopIteration
        self.index = self.index - 1
        return self.data[self.index]
  • 生成器
  1. 利用yield语句，在函数中需要返回值的地方用yield语句替代return，每次对生成器调用next()方法时，会从上次离开的位置恢复执行（会记住上次执行语句时的所有数据值）。
def reverse(data):
    for index in range(len(data)-1, -1, -1):
        yield data[index]
  1. 用生成器表达式
[data[i] for i in range(len(data)-1, -1, -1)]
装饰器

理解装饰器之前，首先要明白在python中函数也是一个对象，也可以被赋值。

def hello():
    print('hello world')
sayHi = hello
del hello
hello() # NameError: name 'hello' is not defined
sayHi() # hello world
  • 需求

在一个函数执行之前做一点其他的事情，比如常见的日志等

  • 实现
def log(func):
    def wrapper(*args, **kw):
        print('call %s():' %func.__name__)
        return func(*args, **kw)
    return wrapper

@log
def run():
    print("hello world")
run() # call run():\n hello world

可见，上面的方法中，log接受函数作为参数，同时，在里面定义的wrapper方法中以函数作为返回值。这样，就可以在函数func被执行之前，做任意操作了。

  • 另一点需求

但是，如果log也想接受一个额外的参数呢？

  • 实现

再套一层外部封装

def log(text):
    def decorator(func):
        def wrapper(*args, **kw):
            print('%s call %s():' %(text, func.__name__))
            return func(*args, **kw)
        return wrapper
    return decorator

@log('iyuge2')
def run():
    print("hello world")
run() # iyuge2 call run():\n hello world

上面方法等价于log('iyuge2')()。

  • 另一个问题

上面那种装饰方法会改变run.__name__的值，除了在装饰器内部重新给func.__name__赋值之外，python提供了另一种新的方法实现。

import functools

def log(func):
    @functools.wraps(func)
    def wrapper(*args, **kw):
        print('call %s():' % func.__name__)
        return func(*args, **kw)
    return wrapper

def log(text):
    def decorator(func):
        @functools.wraps(func)
        def wrapper(*args, **kw):
            print('%s %s():' % (text, func.__name__))
            return func(*args, **kw)
        return wrapper
    return decorator

其实，也就是functools.wraps也用了一层封装，将__name__属性重新改了回来。
