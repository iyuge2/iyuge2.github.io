# iyuge2.github.io

Personal site powered by Jekyll and the Minimal Mistakes remote theme.

## Local preview

```bash
bundle install
bundle exec jekyll serve
```

## Add a post

Create a Markdown file in `_posts/`:

```text
YYYY-MM-DD-title.md
```

Each post should include front matter:

```yaml
---
title: "Post title"
date: 2026-05-06 18:00:00 +0800
categories:
  - notes
tags:
  - jekyll
---
```
